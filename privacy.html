隐私政策

最后更新：2026-06-25

1. 信息收集

我们仅收集提供服务所必需的信息，包括：Telegram 用户名（用于客服沟通）、API 调用日志（用于计费和监控）。我们不会收集您的对话内容。

2. 信息使用

收集的信息仅用于：提供 API 服务、计费、故障排查。我们不会将您的信息出售或分享给第三方。

3. 数据安全

我们采用行业标准的安全措施保护您的数据。API 通信全程使用 TLS 加密。

4. 数据保留

API 调用日志保留 90 天后自动删除。账户信息在账户关闭后保留 30 天。
